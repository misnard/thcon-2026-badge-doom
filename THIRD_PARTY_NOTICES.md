## embeddedDOOM

This badge port is derived from the upstream embeddedDOOM project:

- Repository: https://github.com/cnlohr/embeddedDOOM
- Upstream author: Charles Lohr and contributors
- Purpose here: memory-constrained DOOM engine base, asset baking flow, and
  embedded-oriented rendering/runtime structure

The upstream code was merged into this repository instead of kept as a git
submodule so the badge firmware can be built from a single checkout.
